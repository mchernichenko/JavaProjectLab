1. Как создать проект с нуля
предусловия: забрать подсистемы из Git
- создать навый проект, idea предлагает добавить(импортировать) модули. Под модулем понимаем подсистему, имеющую отдельный Git-репо
- повиноваться IDEA т.е. сконфигурить в Settings->Version Control корневые директории репозитория Git,
  т.к. в данном случае каждый модуль (подсистема) это отдельный Git-репо, а сам корневой каталог проекта не является артифактом Git и его удаляем из настроек
  Итого имеем в одном проекте несколько модулей, которые представляют из себя разные Git-репозитории

2. Навигатор по репозиторию: VCS->Browse VCS Repository ->Show Git Repo Log
   Показывает структуру всего рапозитория с хоролшей возможностью всяких фильтраций и сортировок (в отличие от Git Extentions)

Работа в ветками: переключение на нужную ветку, мерджинг, сравнение, создание(!)

3. Просмотр истории в Git - по проекту / по файлу. Git->Show History (Контекстное меню на проекте) позволяет видеть историю и в рамках каждой истории видеть изменённые файлы и в рамках изменённого файла видеть изменённые строчки (в общем как в Git Extentions)
4. Просмотр построчной информации об изменениях (Annotate). Git->Annotate (контекстное меню из текущего открытого файла) позволяет видеть его историю изменения
