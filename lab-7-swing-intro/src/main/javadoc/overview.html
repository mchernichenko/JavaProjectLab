Всё начинается с окна, т.е создание GUI начинается с создание фрейма, т.е. с new JFrame() или с класса наследника JFrame JFrame - объект, который представляет собой окно на экране в которое помещаются все остальные элементы GUI JFrame по-разному выглядит в зависимости от платформы 2. Долее во фрейм добавляются различные виджеты (кнопки, эдементы управления, графику и пр.). 3. Некоторые виджеты являются ИСТОЧНИКАМИ событий, основываясь на действиях пользователя. Понять это можно по наличию в классе метода add#ТипСобытия#Listener - метод добавления случателя в список подписчиков actionPerformed - метод интерфейса ActionListener вызываемый источником события для всех подписчиков (метод обратного вызова) 4. Чтобы создать свой виджет для рисования создаей класс расширяя JPanel и весь графический код располагаем в методе paintComponent. С помощью этого метод система выдаёт нам холст (Graphics g) на котором можно рисовать. Этом метод никогда самостоятельно не вызывается (через freme.repaint()). 5. Компановка графических элементов. По умолчению фрейм имеет 5 областей: север/юг/запад/восток/центр(по умолчению). В одной области можно разместить только один объект. Но область можно добавить панель, которая в свою очередь такжа будет иметь 5 областей и т.д.
