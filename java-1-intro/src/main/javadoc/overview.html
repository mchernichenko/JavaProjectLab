Заметки для OCA/OCP. 1. Допустимые имена идентификаторов: - начинаться могуть только с буквы, $, _ - с цифры начинатьсы не могут - ограничении на длину нет - нельзя использовать ключевые слова - идентификаторы регистрозависимые int _a; int $c; int ______2_w; int _$; int this_is_a_very_detailed_name_for_an_identifier; The following are illegal (it's your job to recognize why): int :b; int -d; int e#; int .f; int 7g; 2. Java Code Conventions (все объкты в CamelCase) - Классы (обысно имя существительное) и интерфейсы (прилагательное) с большой буквы: Account, Runnable - Методы с маленькой verb-noun: getBalance - Переменые c мальенькой - Константы большими c разлелителями _: MIN_HEIGHT 3. на экзамене нужно помнить размеры примитивных типов и формула расчет диапазона: от -2^(bits-1) до 2^(bits-1)-1 (т.к. 0 считается положительным) byte-8bits, short16, int32, long64, float32, double64 boolean не имеет диапазона, только true/false char16 диапазон только положительный: 0-2^16(65535); Формально тип char целочисленный
